# Changelog

## 0.1.0

Initial release.

- WYSIWYG-like editing: hides markdown syntax on inactive lines, reveals on cursor line
- Sans-serif prose font for body text; monospace for code and tables
- Headings with configurable font-size scaling and bold weight
- Inline formatting: bold, italic, bold+italic (including mixed markers), strikethrough
- Inline code with background highlight
- Links with underlined text
- Blockquotes with vertical bar and italic styling
- Unordered and ordered list bullet/number replacement
- Fenced code blocks with background
- Horizontal rules
- Tables with header styling, alignment row separator, and pipe replacement
- Inline math rendering (LaTeX between `$...$`)
